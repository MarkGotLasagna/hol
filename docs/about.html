About the Project

The History Of Linux Project (HOLP) is an open-source initiative dedicated to illustrating the rich history of Linux through an interactive timeline with colorful cards and images.

What is HOLP?

Each card in our diagram represents a core event in history that contributed to the creation of Linux and what Linux is today. Images placed above the timeline illustrate important figures, logos belonging to companies or organizations that helped directly or passively with the creation of Linux.

Our Goals

  • Illustrate how Linux came to be, from its birth till present day
  • Preserve the history of the Linux kernel's development
  • Be an academic resource for teaching Linux's history
  • Raise awareness of the open-source initiative

How to Contribute

HOLP is an open-source project that welcomes volunteers to:

  • Add, remove or modify cards about historical events
  • Fact check written information, including both cards and dates
  • Suggest artistic changes to the timeline
  • Enhance accessibility and readability
  • Add translations

Visit our GitHub repository to get started with contributions.
